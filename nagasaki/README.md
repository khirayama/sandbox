## 長崎市コンテンツ

長崎県ではなく長崎市。長崎県は周辺コンテンツとしてカバー。
参考: https://yokanavi.com/

## テーマ

- 来る
- 住む
- 移る

## 情報

データとコンテンツに分けて考える。データはストックしていく。
コンテンツはデータ以外の形に。ただコンテンツもストックとフローにはできそうなので、そこはうまく分ける。
データに関してはjson APIか何かで公開できるとよいけど。
